# Changelog

All notable changes to this project will be documented in this file.

This file is automatically updated by the CI/CD pipeline when new versions are released.

## Format

This changelog follows the [Conventional Commits](https://www.conventionalcommits.org/) specification.

- **✨ Features**: New features and enhancements
- **🐛 Bug Fixes**: Bug fixes and corrections
- **⚠️ Breaking Changes**: Changes that break backward compatibility
- **🔧 Other Changes**: Documentation, refactoring, and other changes
